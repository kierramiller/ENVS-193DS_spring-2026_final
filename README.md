# ENVS-193DS_spring-2026_final
Repository for final assignment in ENVS 193DS Spring 2026

## General Information

This repository contains the final assignment for ENVS 193DS (Environmental Data Science), Spring 2026. The assignment has three components:

1. **Research writing** – Evaluating and improving statistical communication in a report about American Avocet (*Recurvirostra americana*) microhabitat use in the San Francisco Bay.
2. **Data analysis** – Investigating how tree height and aacacia ant species identity influence the probability of bird nest occurrence using logistic regression models, based on data from Lujan et al. (2023).
3. **Affective and exploratory visualizations** – Comparing personal data visualizations from earlier assignments, proposing and running an appropriate statistical analysis, and sharing the affective visualization in class.

## Data and File Information

The repository is organized into the following folders:

- `code/`: Contains the Quarto (.qmd) file used to complete the assignment, including all annotated code, analysis, and written responses. Also contains the rendered PDF output.
- `data/`: Contains datasets used in the assignment, including the nest box occupancy dataset from Lujan et al. (2023) and personal data collected for Problem 3.

**Data citation:**
Lujan, E., Nielsen, R., Short, Z., Wicks, S., Nderitu Watetu, W., Malingati Khasoha, L., Palmer, T. M., Goheen, J. R., & Alston, J. M. (2023). Data, code, and metadata for: Symbiotic acacia ants drive nesting behavior by birds in an African savanna [Data set]. Zenodo. https://doi.org/10.5281/zenodo.8373322

Additional files:

- `.Rproj`: R project file for organizing the workspace.
- `.gitignore`: Specifies files that should not be tracked by Git.
